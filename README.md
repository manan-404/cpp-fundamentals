# C++ Fundamentals

A collection of my **Programming Fundamentals (C++)** lab exercises and practice programs from my **1st semester of BS Computer Science**.

This repository covers the core concepts of C++ programming, starting from basic input/output and expressions, progressing through functions, arrays, strings, pointers, and structures.

The purpose of this repository is to document my learning journey, preserve lab work, and provide organized examples for students learning C++ fundamentals.

## How to Run

### Using Dev C++

1. Open Dev C++
2. Open any `.cpp` file
3. Press **F11** to Compile & Run

---

### Using Command Line (g++)

Compile:

```bash
g++ filename.cpp -o filename
```

Run:

```bash
./filename
```

Example:

```bash
g++ palindrome-checker.cpp -o palindrome
./palindrome
```
